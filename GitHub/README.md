## How to Commit/Push to Github

~~~
git clone <repo>     or        git -c http.sslVerify=false clone <repository-name - ssh>

cd <repo path>

code . 

git pull 

git checkout -b <new branch_name>

git status 

git add -A

git commit -m "" 

git push origin [branch]

git pull origin [branch]
~~~
